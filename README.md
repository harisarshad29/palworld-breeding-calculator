# Palworld Breeding Calculator (Rust)

Rust-powered Palworld breeding calculator with API-backed logic and database-style UI.

## Tech Stack

- Rust (`axum`, `tokio`, `tower-http`)
- Frontend: HTML/CSS/JS
- Local static Pal images in `assets/pals/`

## Run

```bash
cargo run
```

Open: <http://127.0.0.1:3000>

## API Endpoints

- `GET /api/bootstrap`
- `POST /api/calculate`
- `GET /api/combinations/:target`
- `GET /api/capture/:target`

## SEO Setup (Rust-managed)

SEO is served by Rust in `src/main.rs`:

- `/` injects meta tags + Open Graph + Twitter + JSON-LD
- `/robots.txt` is generated by Rust
- `/sitemap.xml` is generated by Rust

Set environment variables before deploy:

```bash
BASE_URL=https://yourdomain.com
HOST=0.0.0.0
PORT=3000
```

Release build (recommended for production):

```bash
cargo build --release
./target/release/palworld-breeding-calculator.exe
```

## SEO QA (local)

With the server running (`cargo run`), in another terminal:

```powershell
powershell -ExecutionPolicy Bypass -File scripts/seo-qa.ps1
```

Set `SEO_QA_BASE=https://yourdomain.com` to test a deployed build.

## Production SEO Checklist

- Set `BASE_URL` to live domain in production.
- Verify `https://yourdomain.com/robots.txt` returns 200.
- Verify `https://yourdomain.com/sitemap.xml` includes all key routes.
- Submit sitemap in Google Search Console.
- Inspect `/`, `/palworld-breeding-calculator`, `/pals`, `/map`, `/items`, `/technology`, `/capture-rate` with URL Inspection.
- Old `/breeding-calculator` redirects (308) to `/palworld-breeding-calculator`.
- Confirm canonical and OG tags render server-side in page source.
- Add your verified domain property in Search Console and request indexing for key routes.
- Monitor CTR, average position, and indexing coverage weekly.

## Programmatic SEO Pages

The Rust server now also publishes:

- Dynamic pal pages: `/pal/:name` (example: `/pal/anubis`)
- Guide pages:
  - `/guides/how-to-breed-anubis`
  - `/guides/best-breeding-combos`
  - `/guides/capture-rate-explained`

These routes are included in Rust-generated `sitemap.xml`.
